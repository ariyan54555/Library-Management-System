# 📚 Library Management System

A C++ console-based Library Management System built as a course project for the **Algorithm course** in 4th Semester.

## Features
- Add / Delete Books
- Search Books
- Issue & Return Books
- View All Records

## Technologies Used
- Language: C++
- Concepts: Data Structures & Algorithms

## How to Run
```bash
g++ main.cpp -o library
./library
```
